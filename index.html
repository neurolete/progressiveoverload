01 // Warm-Ups (What to Do + Why)

Get warm first: walk on a treadmill for 5–10 minutes, then follow the 5-minute warm-up video at the top of your training program.

Use the 5–3–3–1 method for:

  • RDLs & any deadlift variation
  • Hip thrusts
  • Leg press
  • Squats & any squat variation
SET 015 reps @ 40–50%
SET 023 reps @ 60%
SET 033 reps @ 70%
SET 041 rep @ 80–90%

Do not skip the single. If rushing, skip one of the 3s before skipping the 1.

All other exercises: 1 solid warm-up set using 60–70% of your first working set for 5–7 reps.

Goal: prime the movement + nervous system — not stretch.

02 // Overload Logic

Did you hit the rep range?

Below Range

Treat as warm-up
Reduce load → repeat

In Range

Perfect execution
Next session: +3–5%

Above Range

Under-loaded
Next session: +5–10%

Rest 2–3 min → repeat set to technical failure (if prescribed)
