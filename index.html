01 // WARM-UPS (WHAT TO DO + WHY)
> GET WARM Walk on the treadmill for 5–10 minutes, then follow the 5-minute warm-up video link located at the top of your training program.
> 5–3–3–1 WARM-UP METHOD Use this method for:
  • RDL and any deadlift variation
  • Hip thrusts
  • Bench press
  • Leg press
  • Squat and any squat variation
SET 01 5 reps @ 40–50% of 1st working set
SET 02 3 reps @ 60% of 1st working set
SET 03 3 reps @ 70% of 1st working set
SET 04 1 rep @ 80–90% of 1st working set
Don’t skip the single. If you’re rushing, skip one of the “3s” before you skip the 1.
> ALL OTHER EXERCISES For all other exercises, 1 solid warm-up set is enough: use 60–70% of your first working set weight for 5–7 reps.
> WHY THIS MATTERS The goal is to get warm + prime the movement pattern and nervous system — not to “stretch.” A stretched muscle isn’t a strong muscle.
